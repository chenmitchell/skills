# üß≠ Model Router for OpenClaw

Self-learning multi-provider model routing. Auto-detects your available models, discovers new ones when you add them, fetches their benchmarks, and updates routing automatically.

Benchmarks are routing tables, not leaderboards.

## Install

```bash
clawhub install chandika/model-router
```

Or manually copy `SKILL.md` to your OpenClaw skills directory.

## How It Works

1. **Detects** which providers and models you have configured
2. **Maintains a registry** ‚Äî `model-registry.json` with benchmark scores, pricing, and routing rules
3. **Discovers new models** ‚Äî when you add a model to OpenClaw, the router detects it, fetches its model card and benchmarks via web search, and tells you where it fits
4. **Recommends** the best routing mode based on what's available
5. **Adapts** ‚Äî "work harder" escalates to Opus, "save money" drops to Gemini/DeepSeek
6. **Hard routes by task** ‚Äî computer use always goes to Claude (72.5% vs GPT's 38.2%), deep reasoning always goes to Opus

## Three Modes

### üèÜ Performance ‚Äî "Work hard"

Opus 4.6 main, Sonnet 4.6 subagents. Best quality. Claude-only.

```json
{
  "agents": {
    "defaults": {
      "model": { "primary": "anthropic/claude-opus-4-6" },
      "subagents": { "model": "anthropic/claude-sonnet-4-6" }
    }
  }
}
```

### ‚öñÔ∏è Balanced ‚Äî "Normal" (recommended)

Sonnet 4.6 main, Gemini 2.5 Pro subagents. Good quality, rate limits survive.

```json
{
  "agents": {
    "defaults": {
      "model": { "primary": "anthropic/claude-sonnet-4-6" },
      "subagents": { "model": "google/gemini-2.5-pro" }
    }
  }
}
```

### üí∞ Economy ‚Äî "Save money"

Gemini 2.5 Pro main, Flash/DeepSeek subagents. Max efficiency.

```json
{
  "agents": {
    "defaults": {
      "model": { "primary": "google/gemini-2.5-pro" },
      "subagents": { "model": "google/gemini-2.5-flash" }
    }
  }
}
```

## Task Overrides (always apply)

| Task | Model | Why |
|------|-------|-----|
| Computer use / browser | Claude | 72.5% vs 38.2% ‚Äî hard rule |
| Deep reasoning / novel problems | Opus 4.6 | 75.2% ARC-AGI-2 ‚Äî uniquely capable |
| Office / finance | Sonnet 4.6 | 1633 Elo ‚Äî beats everything incl. Opus |
| Drafts / summaries | Cheapest | Don't overthink it |

## Adaptive Triggers

| You say | It does |
|---------|---------|
| "work harder" | Suggests Performance mode |
| "save money" | Suggests Economy mode |
| "normal" / "reset" | Reverts to Balanced |
| "use opus for this" | Session-only override |

## Why

v1 solved Claude-only routing. v2 expanded to multi-provider. v2.1 makes it self-learning.

Add a new model to OpenClaw ‚Üí the router detects it, researches its benchmarks, figures out where it beats your current setup, and asks if you want to use it. Remove a model ‚Üí it reroutes affected tasks to the next best option.

One provider or five ‚Äî the router adapts to what you have and keeps adapting as your setup changes.

See [SKILL.md](./SKILL.md) for full benchmark tables, all configs, and routing logic.
