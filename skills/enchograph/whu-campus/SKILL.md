# WHU Campus

获取武汉大学校园新闻。

## 数据获取

### 日期获取

**必须先获取当前日期**：
使用 `session_status` 获取当前日期，用于动态生成搜索关键词和输出日期

### 搜狗微信搜索地址

使用搜狗微信搜索，多次查询不同关键词：

```
https://weixin.sogou.com/weixinwap?type=2&query=搜索关键词
```

### 重点检索学院方向

**汉语言、历史、新传、经管、电信、计算机、网安**是重点检索学院方向。

### 必查分类（多次搜索）

**注意**：所有搜索关键词必须包含当前日期，确保时效性。搜索日期要根据当前日期动态变化。

#### 1. 武汉大学重大新闻
```
武汉大学 新闻 [日期] 
```

#### 2. 武汉大学讲座
```
武汉大学 讲座 [日期] [学院]
武汉大学 学术讲座 [日期] [学院]
武汉大学 学术讲坛 [日期] [学院]
武汉大学 讲座预告 [日期] [学院]
```

#### 3. 武汉大学活动
```
武汉大学 活动 [日期]
武汉大学 学生活动 [日期]
武汉大学 社团活动 [日期]
```

#### 4. 武汉大学通知
```
武汉大学通知 [日期] [学院]
武汉大学公告 [日期] [学院]
武汉大学重要通知 [日期] [学院]
武汉大学重要须知 [日期] [学院]
```

#### 5. 武汉大学全心权益公众号（重点内容来源）
```
武汉大学全心权益 [日期]
全心权益公众号 [日期]
武汉大学学生会 [日期]
```


### 查询策略

- **先获取当前日期**：使用 `session_status` 获取当前日期，提取年份、月份、日期
- **动态生成搜索关键词**：根据当前日期生成搜索关键词
- **使用搜狗微信搜索多次查询**：每个分类至少搜索3-5次不同关键词
- **重点爬取"武汉大学全心权益"公众号的新内容**
- **只输出还未举办的活动**：活动举办日期 >= 当前日期
- **新闻具有时效性**：只输出当前月份的新闻和活动

## 输出格式

```
---

🎓 武大校园新闻（[当前日期]）

【重大新闻】
1. 新闻标题

【讲座】
1. 讲座标题
	[举办单位] [举办日期] [举办地点]

【活动】
1. 活动标题
	[举办单位] [举办日期] [举办地点]

【通知】
1. 通知标题
	[通知单位] [概要]

（全心权益公众号的内容根据类型放在上面几条里面）

---
```

搜索后审核：
- 只输出当前月份的新闻和活动
- 只输出还未举办的活动

注意：
- 有几条放几条，确保新闻是最新的
- 来源是武大各微信公众号
- 分门别类输出
- 重点爬取"武汉大学全心权益"公众号的新内容
- 在活动和讲座下简洁指出举办单位、举办日期、举办地点
- 只输出当前月份的新闻和活动
- 只输出还未举办的活动
- 分割线“---”用“======”代替
- 只输出要求输出的内容，前后不要输出多余的文本、说明、报错
