# JEE Exam Configuration

## JEE Main

- **Total Marks:** 300 (100 per subject)
- **Questions:** 90 (30 per subject: 20 MCQ + 10 Numerical)
- **Duration:** 3 hours
- **Attempts:** 2 per year (January + April)
- **Marking:** +4 correct, -1 wrong (MCQ), +4 correct, 0 wrong (Numerical)
- **Purpose:** NIT, IIIT, GFTI admission + JEE Advanced eligibility

## JEE Advanced

- **Total Marks:** 360 (180 per paper)
- **Papers:** 2 papers, 3 hours each
- **Questions:** Variable (MCQ, numerical, matching, paragraph-based)
- **Eligibility:** Top 2.5 lakh from JEE Main
- **Purpose:** IIT admission only
