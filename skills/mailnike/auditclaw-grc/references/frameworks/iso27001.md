## Cross-Framework Mappings

ISO 27001 controls include mappings to:
- **SOC 2**: Trust Services Criteria (e.g., CC6.1, CC7.4)
- **NIST CSF**: Functions/Categories (e.g., PR.AC-1, DE.CM-4)

### Key Mapping Examples

| ISO 27001 | SOC 2 | NIST CSF | Domain |
|-----------|-------|----------|--------|
| A9.1.1 (Access control policy) | CC6.1 | PR.AC-1 | Access |
| A12.4.1 (Event logging) | CC4.1, CC7.2 | DE.AE-3, PR.PT-1 | Monitoring |
| A16.1.5 (Incident response) | CC7.4 | RS.RP-1, RS.MI-1 | Incident |
| A12.6.1 (Vulnerability mgmt) | CC3.2, CC6.6 | DE.CM-8, RS.MI-3 | Operations |
| A18.1.4 (Privacy/PII) | C1.1 | PR.DS-1 | Compliance |
