# Open Notebook Integration Skill

An OpenClaw skill for integrating agents with open-notebook (NotebookLM alternative).

## Features

- Local AI research assistant integration
- Thematic notebooks for different topics
- Query and save capabilities
- Works with Ollama (free, local models)

## Quick Start

1. Install Docker Desktop and Ollama
2. Run `docker compose up -d` in open-notebook folder
3. Load the scripts in this skill
4. Start saving knowledge!

## Files

- `SKILL.md` - Main documentation
- `open-notebook-automation.ps1` - Core functions
- `on-query-handbook.ps1` - Query examples
- `create-notebooks.ps1` - Create thematic notebooks
