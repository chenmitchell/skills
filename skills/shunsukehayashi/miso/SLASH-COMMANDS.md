# MISO Slash Commands

Issue-comment commands:

- `/agent analyze`
- `/agent execute`
- `/agent review`
- `/agent status`
- `/task-start`
- `/task-plan`
- `/task-close`

All status messages are plain text for mobile-friendly readability.

## Telegram menu commands

- /start : Open mission quick-start
- /plan : Show last task plan template
- /close : Show closeout template
- /status : Show live mission status
- /task_start : Alias to /task-start format
- /task_plan : Alias to /task-plan format
- /task_close : Alias to /task-close format
