# 인텐트 라우터 (9개 모드)

> 공통 분류 규칙: [`../../common/intent_router_base.md`](../../common/intent_router_base.md)

## 분류 규칙
사용자 요청을 아래 9개 인텐트 중 하나로 분류한다.
복합 요청은 주 인텐트 + 보조 인텐트로 처리한다.
파라미터 부족 시 1회 질문, 추측 가능하면 추측 후 명시.

---

## 인텐트 목록

### 1. Pension Estimate (국민연금 예상 수령액)
- **표현:** "국민연금 얼마 받아?", "예상 수령액 계산해줘", "연금 얼마나 나와?"
- **산출물:** 가입 기간/소득 기반 예상 수령액 (내장 계산식 + API)
- **최소 데이터:** 현재 나이 또는 가입 기간
- **깊이:** Flash only
- **처리:** 내장 계산식 우선 → 국민연금 API 보조

### 2. Pension Guide (국민연금 제도)
- **표현:** "국민연금 어떻게 가입해?", "추납 하면 유리해?", "반납이 뭐야?", "크레딧 제도"
- **산출물:** 가입·납부·크레딧·특수 제도 안내
- **최소 데이터:** 없음 (상황 파악 후 맞춤)
- **깊이:** Flash + Deep-Dive 요청 시

### 3. IRP vs Savings (세액공제 연금 비교)
- **표현:** "IRP가 나아? 연금저축이 나아?", "세액공제 연금 추천", "DC형 어떻게 해?"
- **산출물:** IRP vs 연금저축 절세 효과 비교 + 상황별 추천
- **최소 데이터:** 없음 (소득 수준 있으면 세액공제율 정확하게)
- **깊이:** Flash + Deep-Dive 요청 시
- **연계:** tax-guide 스킬 (세액공제 상세)

### 4. Retirement Sim (노후 자금 시뮬레이터)
- **표현:** "노후에 얼마 필요해?", "은퇴하려면 얼마 모아야 해?", "노후 준비 됐어?"
- **산출물:** 월 생활비 기반 필요 자산 역산 + 현재 준비 현황 갭 분석
- **최소 데이터:** 희망 은퇴 나이 또는 목표 월 생활비
- **깊이:** Flash + Deep-Dive 요청 시

### 5. Severance Pay (퇴직금)
- **표현:** "퇴직금 얼마야?", "퇴직금 계산해줘", "중간정산 해도 돼?", "DC형 퇴직금"
- **산출물:** 퇴직금 계산 + 수령 방식 + 세금(퇴직소득세)
- **최소 데이터:** 근속 연수, 최근 3개월 평균 임금(선택)
- **깊이:** Flash + Deep-Dive 요청 시

### 6. Basic Pension (기초연금)
- **표현:** "기초연금 얼마야?", "기초연금 대상이야?", "노인연금"
- **산출물:** 수급 조건 + 급여액 + 신청 방법
- **최소 데이터:** 없음 (나이 있으면 대상 여부 확인)
- **깊이:** Flash only
- **연계:** welfare-guide 스킬 (노인 복지 상세)

### 7. FIRE (조기 은퇴)
- **표현:** "조기 은퇴 하고 싶어", "FIRE족 되려면?", "몇억 있으면 은퇴해?", "파이어 계산"
- **산출물:** 4% 룰 기반 필요 자산 + 달성 기간 시뮬레이션
- **최소 데이터:** 월 생활비 또는 목표 은퇴 나이
- **깊이:** Flash + Deep-Dive 요청 시

### 8. Retirement Portfolio (은퇴 포트폴리오)
- **표현:** "은퇴 포트폴리오 짜줘", "노후 자산배분", "TDF 뭐가 좋아?", "연금 투자"
- **산출물:** finance-portfolio-counseling 스킬로 위임 (은퇴 계획 인텐트)
- **최소 데이터:** 나이, 은퇴까지 남은 기간
- **깊이:** finance-portfolio-counseling 스킬 참조

### 9. Beginner Guide (입문)
- **표현:** "연금 처음인데", "국민연금 vs 사적연금", "연금 종류 알려줘"
- **산출물:** 한국 연금 3층 체계 입문 Flash (초보자 모드 강제)
- **최소 데이터:** 없음
- **깊이:** Flash only
