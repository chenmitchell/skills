# 출처 신뢰도 티어

> 공통 Tier 구조 + 인용 형식: [`../../common/source_tiers_base.md`](../../common/source_tiers_base.md)

## Tier 1 — 최고 신뢰도 (공식)
- **국민연금공단** (nps.or.kr) / **내연금** (csa.nps.or.kr) — 국민연금 공식 정보
- **금융감독원** (fss.or.kr) — IRP·퇴직연금 규정
- **고용노동부** (moel.go.kr) — 퇴직급여보장법
- **법제처** (law.go.kr) — 국민연금법, 근로자퇴직급여보장법
- **기획재정부** — IRP·연금저축 세제 혜택 기준

## Tier 2 — 높은 신뢰도
- **금융투자협회** (kofia.or.kr) — TDF·연금펀드 정보
- **미래에셋투자와연금센터** — 연금 투자 리서치
- **한국경제** / **매일경제** 연금 섹션

## Tier 3 — 보강용
- 각 금융사(삼성증권·미래에셋·한투증권 등) 연금 가이드

## 금지 출처
- 개인 블로그·유튜브 재테크 채널 → 인용 금지
- 금융상품 광고성 콘텐츠 → 인용 금지

## 인용 형식
```markdown
# 계산 결과
(내장 계산식: 소득대체율 X%, 가입기간 Y년 기준 추정)
(4% 룰 기반 추정치 — 실제와 차이 있을 수 있음)

# 공식 출처
[국민연금공단](https://www.nps.or.kr)
[내연금 조회](https://csa.nps.or.kr)

# 추정
"추정치" 명시 필수 + 정확한 확인 경로 안내
```
