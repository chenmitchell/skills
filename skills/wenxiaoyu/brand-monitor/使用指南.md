## 快速开始

### 第一次使用

**步骤 1：执行测试监控**

在 Telegram/WhatsApp 对 OpenClaw 说：

```
执行品牌监控
```

**步骤 2：等待报告**

3-5 分钟后，你会在飞书收到第一份监控报告。

**步骤 3：查看报告**

报告包含：
- 📊 总览统计
- 🔥 热门提及 Top 5
- 📱 平台分布
- ⚠️ 需要关注的内容

## 日常使用

### 每日监控

系统会自动在每天早上 9:00 执行监控并推送报告到飞书。

**你需要做的：**

1. **查看报告**（5 分钟）
   - 浏览总览统计
   - 查看热门提及
   - 关注汽车媒体大V的评价
   - 关注警报信息

2. **处理负面提及**（如有）
   - 点击链接查看原文
   - 评估影响程度（特别关注续航、充电、安全等问题）
   - 制定回应策略
   - 及时回复用户

3. **记录洞察**
   - 用户关心什么？（续航、充电、智驾？）
   - 有什么新问题？
   - 竞品动态如何？
   - 汽车媒体如何评价？

### 手动触发监控

如果需要立即查看最新情况：

```
执行品牌监控
```

或指定时间范围：

```
搜索过去 6 小时的品牌提及并分析
```

### 实时警报

系统每小时自动检查一次警报。

**收到警报时：**

1. **立即查看**
   - 警报会标注紧急程度（1-5级）
   - 5级需要立即处理
   - 4级需要当天处理

2. **评估影响**
   - 查看作者影响力
   - 查看传播范围
   - 评估潜在影响

3. **快速响应**
   - 使用建议的回复模板
   - 或自行制定回应
   - 及时发布回应

### 趋势分析

**每周查看一次趋势：**

```
分析过去 7 天的品牌趋势
```

**每月查看一次长期趋势：**

```
分析过去 30 天的品牌趋势
```

## 报告解读

### 总览统计

```
📈 总览
总提及数: 47
😊 正面: 32 (68%)
😐 中性: 12 (26%)
😞 负面: 3 (6%)
```

**如何解读：**

- **总提及数**：品牌曝光度指标
  - 增长：品牌知名度提升
  - 下降：需要加强营销

- **情感分布**：品牌声誉指标
  - 正面 > 60%：健康状态
  - 正面 40-60%：需要关注
  - 正面 < 40%：需要改进

- **负面占比**：风险指标
  - < 10%：正常范围
  - 10-20%：需要关注
  - > 20%：需要立即行动

### 情感分数

| 分数范围 | 标签 | 含义 | 行动 |
|---------|------|------|------|
| 0.5 ~ 1.0 | 非常正面 | 用户非常满意 | 放大传播 |
| 0.2 ~ 0.5 | 偏正面 | 用户基本满意 | 保持现状 |
| -0.2 ~ 0.2 | 中性 | 仅提及，无明确态度 | 观察 |
| -0.5 ~ -0.2 | 偏负面 | 用户有不满 | 关注并改进 |
| -1.0 ~ -0.5 | 非常负面 | 用户非常不满 | 立即处理 |

### 影响力分数

| 分数范围 | 等级 | 含义 | 优先级 |
|---------|------|------|--------|
| 0-100 | 低 | 影响范围小 | 低 |
| 100-500 | 中 | 有一定影响 | 中 |
| 500-1000 | 高 | 影响较大 | 高 |
| 1000+ | 极高 | 可能病毒式传播 | 紧急 |

**影响力计算公式：**
```
影响力 = 粉丝数×0.3 + 点赞数×0.25 + 转发数×0.25 + 评论数×0.2
```

### 警报级别

| 级别 | 图标 | 含义 | 响应时间 |
|------|------|------|----------|
| 5级 | 🔴 | 危机事件 | 立即（1小时内） |
| 4级 | 🟠 | 严重负面 | 当天 |
| 3级 | 🟡 | 中等负面 | 24小时内 |
| 2级 | 🔵 | 轻微负面 | 本周内 |
| 1级 | ⚪ | 仅供参考 | 无需行动 |

## 最佳实践

### 1. 每日例行检查（10 分钟）

**早上 9:15（收到报告后）：**

- [ ] 查看总览统计
- [ ] 浏览 Top 5 提及
- [ ] 检查是否有警报
- [ ] 记录关键洞察

**如有负面提及：**

- [ ] 评估严重程度
- [ ] 制定回应策略
- [ ] 分配处理责任人
- [ ] 设置跟进提醒

### 2. 每周深度分析（30 分钟）

**每周一早上：**

```
分析过去 7 天的品牌趋势
```

**分析重点：**

- 提及数量变化趋势
- 情感分布变化
- 新出现的话题
- 用户关注点变化
- 竞品动态

**输出：**

- 周报 PPT（给团队）
- 改进建议清单
- 下周行动计划

### 3. 每月战略复盘（1 小时）

**每月初：**

```
生成上月的品牌监控月度报告
```

**复盘内容：**

- 月度数据总结
- 关键事件回顾
- 成功案例分析
- 失败教训总结
- 下月策略调整

### 4. 负面提及处理流程

**发现负面提及 →**

1. **评估（5 分钟）**
   - 查看原文和上下文
   - 评估影响范围
   - 判断是否属实

2. **决策（10 分钟）**
   - 是否需要回应？
   - 公开回应还是私下沟通？
   - 谁来回应？

3. **行动（30 分钟）**
   - 起草回应内容
   - 内部审核
   - 发布回应

4. **跟进（持续）**
   - 监控后续反应
   - 评估回应效果
   - 必要时调整策略

### 5. 数据驱动决策

**用监控数据指导：**

- **产品改进**
  - 用户最常抱怨什么？（续航、充电、智驾？）
  - 用户最喜欢什么功能？
  - 有什么新需求？
  - 与竞品相比有什么差距？

- **营销策略**
  - 哪些话题引起共鸣？（续航、智驾、空间？）
  - 哪些渠道效果最好？（小红书、微博、汽车之家？）
  - 什么时间发布最佳？
  - 汽车媒体关注什么？

- **客服优化**
  - 常见问题是什么？
  - 响应速度如何？
  - 满意度如何？
  - 充电、售后服务体验如何？
