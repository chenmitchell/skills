# 快速发布到 ClawHub

## 第一步：获取 ClawHub API Token

⚠️ **重要：使用 ClawHub API Token，不是 GitHub Token！**

1. 访问：https://clawhub.ai

2. 使用 GitHub 登录

3. 进入 Settings → API Tokens
   
   直接访问：https://clawhub.ai/settings/tokens

4. 创建新 Token：
   - 点击 "Create Token"
   - 名称：`CLI Token`
   - 权限：选择 `publish`
   - 点击 "Create"

5. **复制 token**（只显示一次！保存好）

## 第二步：登录 ClawHub

在命令行中运行：

```bash
clawhub login --token YOUR_CLAWHUB_API_TOKEN
```

⚠️ 注意：使用的是 ClawHub 的 API Token，不是 GitHub Personal Access Token！

验证登录：

```bash
clawhub whoami
```

应该显示你的 GitHub 用户名。

## 第三步：发布 Skill

```bash
cd brand-monitor-skill

clawhub publish . --slug brand-monitor --name "Brand Monitor for New Energy Vehicles" --version 1.1.0
```

## 第四步：验证发布

```bash
clawhub search "brand monitor"
clawhub inspect brand-monitor
```

## 完成！

用户现在可以通过以下命令安装：

```bash
npx clawhub install brand-monitor
```

## 如果遇到问题

### "Unauthorized" 错误

❌ 你可能使用了 GitHub Token  
✅ 必须使用 ClawHub API Token（从 https://clawhub.ai/settings/tokens 获取）

### "Account must be at least 1 week old"

你的 GitHub 账号需要至少 1 周才能发布到 ClawHub。

### "Skill already exists"

如果 slug 已被占用，使用不同的名称：

```bash
clawhub publish . --slug brand-monitor-nev --version 1.1.0
```

### 其他问题

查看详细文档：[发布到ClawHub步骤.md](发布到ClawHub步骤.md)
