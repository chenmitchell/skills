# OpenClaw 品牌监控 - 5 分钟快速开始

## 🎯 目标

5 分钟内完成安装和配置，收到第一份品牌监控报告。

---

## 步骤 1：准备飞书 Webhook（2 分钟）

### 方法：创建群聊机器人

1. **打开飞书群聊**
   - 选择一个群聊（或创建新群）
   - 点击右上角 "..." → "设置"

2. **添加机器人**
   - 选择 "群机器人" → "添加机器人"
   - 选择 "自定义机器人"
   - 名称：`品牌监控`
   - 点击 "添加"

3. **复制 Webhook**
   - 复制显示的 Webhook 地址
   - 格式：`https://open.feishu.cn/open-apis/bot/v2/hook/...`
   - 保存到记事本

4. **测试 Webhook**
   ```bash
   curl -X POST "你的Webhook地址" \
     -H "Content-Type: application/json" \
     -d '{"msg_type":"text","content":{"text":"测试成功"}}'
   ```
   
   如果群里收到消息，说明成功！

## 步骤 2：安装 Skill（1 分钟）

### 方式 A：从 ClawHub 安装（最简单）

```bash
# 1. 安装 ClawHub CLI（如果还没安装）
npm i -g clawhub

# 2. 安装 skill
npx clawhub install brand-monitor

# 3. 创建配置文件
cd ~/.openclaw/skills/brand-monitor
cp config.example.json config.json
nano config.json  # 编辑配置
```

### 方式 B：手动安装

```bash
# 1. 下载 Skill
cd ~
git clone https://github.com/your-repo/brand-monitor-skill.git
cd brand-monitor-skill

# 2. 创建目标目录
mkdir -p ~/.openclaw/skills/brand-monitor/prompts

# 3. 只复制必需的文件
cp SKILL.md ~/.openclaw/skills/brand-monitor/
cp config.example.json ~/.openclaw/skills/brand-monitor/
cp prompts/*.md ~/.openclaw/skills/brand-monitor/prompts/

# 4. 验证安装
ls -la ~/.openclaw/skills/brand-monitor/
```

预期看到：
```
SKILL.md              # Skill 元数据（符合 AgentSkills 规范）
config.example.json   # 配置示例
prompts/              # 提示词目录
  monitor.md
  alert.md
  analyze-trend.md
```

## 步骤 3：配置 Skill（1 分钟）

### 创建配置文件

```bash
cd ~/.openclaw/skills/brand-monitor
nano config.json
```

### 粘贴以下内容

```json
{
  "brand_name": "你的品牌名",
  "brand_aliases": ["车型1", "车型2"],
  "exclude_keywords": ["招聘", "代理", "二手车"],
  "platforms": [
    "xiaohongshu",
    "weibo",
    "autohome",
    "dongchedi",
    "yiche",
    "zhihu",
    "tieba",
    "douyin"
  ],
  "monitor_hours": 24,
  "min_engagement": 10,
  "negative_threshold": -0.5,
  "viral_threshold": 5000,
  "feishu_webhook": "粘贴你的Webhook地址",
  "report_language": "zh-CN",
  
  "industry_specific": {
    "focus_keywords": [
      "续航", "充电", "智能驾驶", "辅助驾驶",
      "电池", "安全", "空间", "舒适性"
    ],
    "kol_min_followers": 100000,
    "media_accounts": [
      "汽车之家", "懂车帝", "易车网",
      "新出行", "电动邦", "42号车库"
    ]
  }
}
```

### 保存文件

```
Ctrl + O (保存)
Enter (确认)
Ctrl + X (退出)
```

## 🐛 遇到问题？

### 问题 1：没有收到飞书消息

**检查 Webhook：**
```bash
curl -X POST "你的Webhook" \
  -H "Content-Type: application/json" \
  -d '{"msg_type":"text","content":{"text":"测试"}}'
```

### 问题 2：Skill 未加载

**重新加载：**
```bash
openclaw skills reload
openclaw skills list
```

### 问题 3：搜索无结果

**检查品牌名：**
```bash
cat ~/.openclaw/skills/brand-monitor/config.json | grep brand_name
```

确保品牌名称正确。

### 问题 4：OpenClaw 未运行

**检查状态：**
```bash
openclaw doctor
ps aux | grep openclaw
```

**重启：**
```bash
openclaw restart
```
