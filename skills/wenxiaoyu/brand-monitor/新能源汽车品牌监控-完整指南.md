### 3. 配置说明

**品牌信息：**
- `brand_name`: 主品牌名（如"理想"、"蔚来"、"小鹏"）
- `brand_aliases`: 包含所有车型名称（如"理想L9"、"蔚来ET7"）

**排除关键词：**
- 过滤招聘、代理等无关信息
- 排除二手车、事故车等负面关联

**行业特定配置：**
- `focus_keywords`: 新能源汽车关注的核心话题
- `kol_min_followers`: KOL最低粉丝数（10万+）
- `media_accounts`: 重点关注的汽车媒体

## 🔧 第二步：配置 OpenClaw 飞书集成

### 方式 A：使用 OpenClaw 内置飞书集成（推荐）

既然你已经集成了飞书到 OpenClaw，直接使用 `message` 工具即可。

#### 1. 验证飞书集成

```bash
# 检查 OpenClaw 配置
openclaw config show | grep feishu

# 或查看配置文件
cat ~/.openclaw/openclaw.json | grep -A 5 feishu
```

#### 2. 测试飞书消息

在 Telegram/WhatsApp 对 OpenClaw 说：

```
发送一条测试消息到飞书
```

或使用 CLI：

```bash
openclaw message send --channel=feishu --text="测试消息"
```

如果收到消息，说明集成成功！

### 方式 B：配置飞书群机器人（备用）

如果需要发送到特定群聊：

#### 1. 创建群机器人

1. 打开飞书群聊
2. 点击右上角 "..." → "设置"
3. 选择 "群机器人" → "添加机器人"
4. 选择 "自定义机器人"
5. 名称：`品牌舆情监控`
6. 复制 Webhook 地址

#### 2. 配置到 OpenClaw

编辑 `~/.openclaw/openclaw.json`：

```json
{
  "integrations": {
    "feishu": {
      "enabled": true,
      "webhooks": {
        "brand_monitor": "https://open.feishu.cn/open-apis/bot/v2/hook/你的Webhook"
      }
    }
  }
}
```

## ⏰ 第三步：配置定时任务（核心）

### 方式 1：通过对话配置（最简单）

在 Telegram/WhatsApp 对 OpenClaw 说：

```
请帮我设置品牌舆情监控的定时任务：

1. 每天早上 9:00 执行完整的品牌监控
   - 搜索所有平台的品牌提及
   - 分析情感和影响力
   - 生成报告
   - 发送到飞书

2. 每小时检查一次紧急警报
   - 搜索最近1小时的提及
   - 检测负面提及和病毒式传播
   - 如有紧急情况立即推送到飞书

3. 每周一早上 10:00 生成周报
   - 分析过去7天的趋势
   - 生成趋势分析报告
   - 发送到飞书

请立即设置这些定时任务，并确认设置成功。
```

OpenClaw 会自动：
1. 创建 cron 任务
2. 配置执行时间
3. 设置推送渠道
4. 确认任务已激活

### 方式 2：通过配置文件（精确控制）

#### 1. 编辑 OpenClaw 配置

```bash
nano ~/.openclaw/openclaw.json
```

#### 2. 添加 Cron 任务

```json
{
  "cron": {
    "enabled": true,
    "jobs": [
      {
        "id": "brand-monitor-daily",
        "name": "品牌舆情监控-每日",
        "description": "每天早上9点执行品牌监控",
        "schedule": "0 9 * * *",
        "enabled": true,
        "action": {
          "type": "skill",
          "skill": "brand-monitor",
          "prompt": "monitor",
          "variables": {}
        },
        "notification": {
          "channel": "feishu",
          "on_success": true,
          "on_failure": true
        }
      },
      {
        "id": "brand-monitor-alert",
        "name": "品牌舆情监控-警报",
        "description": "每小时检查紧急警报",
        "schedule": "0 * * * *",
        "enabled": true,
        "action": {
          "type": "skill",
          "skill": "brand-monitor",
          "prompt": "alert",
          "variables": {
            "alert_hours": 1
          }
        },
        "notification": {
          "channel": "feishu",
          "on_success": false,
          "on_failure": true
        }
      },
      {
        "id": "brand-monitor-weekly",
        "name": "品牌舆情监控-周报",
        "description": "每周一早上10点生成周报",
        "schedule": "0 10 * * 1",
        "enabled": true,
        "action": {
          "type": "skill",
          "skill": "brand-monitor",
          "prompt": "analyze-trend",
          "variables": {
            "days": 7
          }
        },
        "notification": {
          "channel": "feishu",
          "on_success": true,
          "on_failure": true
        }
      }
    ]
  }
}
```

#### 3. 保存并重启

```bash
# 保存文件
Ctrl + O, Enter, Ctrl + X

# 重启 OpenClaw
openclaw restart

# 验证 Cron 任务
openclaw cron list
```

### 方式 3：使用系统 Cron（备用）

如果 OpenClaw Cron 不可用，使用系统 cron：

```bash
# 编辑 crontab
crontab -e

# 添加任务
# 每天早上 9:00 执行监控
0 9 * * * cd ~/.openclaw && openclaw skill run brand-monitor monitor

# 每小时检查警报
0 * * * * cd ~/.openclaw && openclaw skill run brand-monitor alert --hours=1

# 每周一早上 10:00 生成周报
0 10 * * 1 cd ~/.openclaw && openclaw skill run brand-monitor analyze-trend --days=7
```
