## 🎯 使用方法

### 第一步：确认必需文件已安装

```bash
cd ~/.openclaw/skills/brand-monitor

# 检查必需文件
ls -la

# 应该看到（只需要这些文件）：
# SKILL.md              - Skill 元数据
# config.example.json   - 配置示例
# prompts/              - 提示词目录
#   monitor.md          - 每日监控
#   alert.md            - 实时警报
#   analyze-trend.md    - 趋势分析
```

### 第二步：创建配置文件

```bash
# 复制配置示例
cp config.example.json config.json

# 编辑配置
nano config.json
```

粘贴以下内容：

```json
{
  "brand_name": "你的品牌名",
  "brand_aliases": ["车型1", "车型2"],
  "exclude_keywords": ["招聘", "代理", "二手车"],
  "platforms": [
    "xiaohongshu",
    "weibo",
    "autohome",
    "dongchedi",
    "yiche",
    "zhihu",
    "tieba",
    "douyin"
  ],
  "monitor_hours": 24,
  "min_engagement": 10,
  "negative_threshold": -0.5,
  "viral_threshold": 5000,
  "report_language": "zh-CN",
  
  "industry_specific": {
    "focus_keywords": [
      "续航", "充电", "智能驾驶", "辅助驾驶",
      "电池", "安全", "空间", "舒适性",
      "性价比", "售后", "质量", "故障"
    ],
    "kol_min_followers": 100000,
    "media_accounts": [
      "汽车之家", "懂车帝", "易车网",
      "新出行", "电动邦", "42号车库"
    ]
  }
}
```

### 第三步：重启 OpenClaw

```bash
openclaw restart
```

### 第四步：测试

```bash
# 测试每日监控
openclaw skill run brand-monitor monitor

# 测试警报检测
openclaw skill run brand-monitor alert --hours=1

# 测试趋势分析
openclaw skill run brand-monitor analyze-trend --days=7
```

### 第五步：配置定时任务

在 Telegram/WhatsApp 对 OpenClaw 说：

```
请帮我配置品牌舆情监控定时任务：
1. 每天早上9点执行监控并发送到飞书
2. 每小时检查警报
3. 每周一早上10点生成周报

请立即设置。
```
